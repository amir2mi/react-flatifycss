## Show / hide

To toggle the alert visibility state, set the `show` to `true` or `false`. Note that `show={false}` will unmount the alert component.

<div class="funky-spacer" />

## Icon

Alerts can icons, add the `<AlertIcon />` component inside `<Alert />`.

<Canvas>
  <Story id="components-alert--icon" />
</Canvas>

## Floating close button

Alert comes with an `<AlertCloseButton />` component which reserves its parent space, another way is to make it comes over the content and be displayed while focued or the parent is hovered, to do so add the `floating` prop to the `<AlertCloseButton />` component.

  <Story id="components-alert--floating-close-button" />
</Canvas>

## Animation

To changed the default dismiss aniamtion use the `animation` prop and set your custom animation like `my-custom-animation 0.4s ease`.
