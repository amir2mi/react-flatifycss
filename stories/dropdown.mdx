# Dropdown

## Default

Note: ALWAYS pass the `id` prop to dropdowns, because it uses this ID internally for accessibility reasons.

## Menu items

To have a dropdown which contains menu items, add the `isMenu={true}` prop to `<DropdownBody />`.  
Now you can add your menu items inside that component like `<li className="menu-item heading">My Account</li>`.

<Canvas>
  <Story id="components-dropdown--menu-items" />
</Canvas>

## Placement

It is possible to change the direction where the dropdown is displayed, use the `placement` with `<Dropdown />` component.  
Possible values are: `top`, `bottom`, `left`, `right`.

<Canvas>
  <Story id="components-dropdown--placement" />
</Canvas>

## Hoverable

If the `isHoverable={true}` prop is set for `<Dropdown />` component, when the user hovers the dropdown button, the dropdown will be displayed (just like a tooltip).

<Canvas>
  <Story id="components-dropdown--hoverable" />
</Canvas>

### Hover show/hide delay

Changing the amount of time that should be passed to display or hide the dropdown is possible with these two props: `hoverShowDelay` (default: 200), `hoverHideDelay` (default: 400), you can set them to the `<Dropdown />` component.

<div class="funky-spacer" />

## Auto close

The dropdown component gives you the ability to limit when it should be closed, for example when the user clicks outside, inside, both or neither.  
For this purpose, use the `autoClose` for the `<Dropdown />` component.  
Possible values are: `true`, `false`, `outside`, `inside`.

<Canvas>
  <Story id="components-dropdown--auto-close" />
</Canvas>

## Offset

## Dropdown button

### Reset style

Set the `buttonStyle={false}` to `<DropdownButton />`, so the button will not have its style anymore and will be like texts.

<div class="funky-spacer-sm" />

### Hide arrow

To hide the dropdown button component arrow, set the `hasArrow={false}` to `<DropdownButton />`.

<div class="funky-spacer-sm" />
