# Select

<div class="funky-spacer" />

## Default

Pass an array of options with `options` prop, use `value` with `onChange` to update data.  
If you do not set the value, it will be set internally as an uncontrolled component by default, you can use `onChange` event to get the updated value, it returns selected options and event.

## Grouped

It is possible to organize options with groups, just add the group items with `options` inside your first level items.
